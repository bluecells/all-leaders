## Style de Code

- Utilise Tailwind CSS pour tous les composants UI.
- Les tableaux doivent être responsives (stackables en mobile)
- Ne jamais utiliser de bibliothèque CSS externe autre que Tailwind.
- N'utilise pas la capitalisation à l'américaine
- Respecte strictement les règles typographiques françaises (espaces, guillemets, majuscules)
